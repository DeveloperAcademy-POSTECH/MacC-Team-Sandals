## ⛳️ Features
### 독립서점 정보 제공
- 현재 위치를 기준으로 가까운 서점 정보 제공
- 서점의 상세 정보(연락처, SNS, 운영시간, 서점 설명글, 위치) 제공
- 지역별 서점 정보 제공
- 서점 북마크 기능
<br/>

### 큐레이션 기능 제공
- 서점/도서에 대한 큐레이션 작성 기능 제공
- 작성한 큐레이션을 기준으로 메인페이지의 큐레이션 업데이트
- 큐레이션 좋아요, 댓글 기능 공
